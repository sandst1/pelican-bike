# pelican-bike
This is a game about a bike-riding pelican, created with a REAP-pruned DeepSeek 4 Flash on a single DGX Spark.

Model: https://huggingface.co/0xSero/DeepSeek-V4-Flash-180B

## setup
```
python -m venv venv
pip install -r requirements.txt
source venv/bin/activate
python pelican_bike.py
```
